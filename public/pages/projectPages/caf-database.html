##Description *** CAF Animal Database is a program created to store the details of animals under the supervision of the University of Guelph's Central Animal Facility. **The following details of the animals can be stored in this database:** * Picture * Physical descriptions (Weight, colour, breed, ...) * Identification information * Fecal history * Vaccination history **The features of the database include:** * Search based on any detail * Sort based on any detail * Print ID cards ##Screenshots ![Database main screen](dist/assets/imgs/caf_main.gif) ![Database search & sort](dist/assets/imgs/caf_search.gif)
